# Инструкция создания проекта в Noodl

* Создать проект, используя шаблон Noodl "Hello World". Название проекта не имеет технического значения.
* Удалить все, кроме компоненты App. Удалить содержимое компоненты App. Noodl должен ругануться на отсутсвие корневого элемента.
* В настройках проекта установить:
    * Head Code:
    ```html
    <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
    <script>
        window.R = {
            states: {},
            env: {},
            params: {},
            libs: {},
            utils: {}
        }
    </script>
    ```
    * Title: заголовок для браузера.
    * URL Path type: Hash.
    * Body Scroll: выключить.
    * Отключить эксперементальные настройки Noodl.
* Открыть папку проекта, перейти в noodl_modules, удалить material-icons, скопировать пакеты Rolder Kit:
    * Обязательные: app, mantine, mantine-old, libs, utils.
    * Не обязательные:
        * data - для использования бекенда.
        * pdf - просмотр и генерация PDF.
        * gantt - построение графика Ганта.
        * charts - построение аналитических графиков.
* Перезайти в проект, вернутся в настройки проекта и установить в появившейся группе Rolder:
    * Project name - техническое название проекта. Должно соответсвовать названию проекта в бекенде. Обязательный параметр.
    * Project version - версия проекта. Обязательный параметр.
    * Project defaults - дефолтные значения. Используются компонентами, в компонентах описано, что нужно устанавливать. Опционально.
    * Mantine theme - [тема Mantine](https://v6.mantine.dev/theming/theme-object/). Опционально.
* Добавить компоненту RK App в App. Сделать App корневой компонентой - App в навигации, три точки, Make home.
* После Make home, App руганется, что не выбрана версия. Выбрать версию.
* Добавить в App компонету Mantine.
* Добавить в Mantine Data, если нужно.
