# App

Корневая компонента Rolder Kit. Чем занимается:
* Проставляет параметры проектв в window.R    
* Включает дебагер, если в адресной строке указано ?debug= 1 или 2.
* Показывает загрузчик.
* Управляет цветовой схемой проекта.
* Проверяет изменение версии и форсирует перезагрузку страницы для обновления кеша браузера.

## Характеристики

* Сolor scheme:
  * Light - светлая тема и дефолтное значение.
  * Dark - темная тема.
  * Auto - тема системы. В этом режиме управлять темой нельзя.
* Когда не Auto, можно:
  * Отправить сигнал Set color scheme и Color scheme (`light` или `dark`) для установки темы вручную. Тема сохранится в Local storage браузера и восстанавливается при загрузке.
  * Отправить сигнал Toggle color scheme для переключения темы. Тема так же сохраниться и будет восстанавливаться при загрузке.
  * В обоих случаях на выходе срабатывает сигнал Color scheme changed и меняется Color scheme на выходе.
* Ограничение - т.к. у нас все приложения рендерятся только на клиенте, при загрузке нет темы, экран белый. Когда загрузился Rolder Kit, тема преключится.
