# Mantine

Корневая компонента для всех компонент, основанных на билиотеке Mantine. Чем занимается:
* Создает контекст для всех дочерних компонентов Mantine.
* Создает контекст для всплывающих сообщений.
* Устанавливает Mantine theme из настроек проекта.
* Устанавливает локаль для компонентов календаря.

## Характеристики

* Notifications position: расположение всплывающих сообщений.
