# Mantine theme

Тема Mantine. Управляет дефолтными значениями компонент. Задается в Head Code, в R.params

Документация [Mantine](https://v6.mantine.dev/theming/default-props/).

Елси в mantineTheme задан параметр, компонента для которой он задан выставит такой параметр как дефолтный в панели параметров Noodl.

Елси в mantineTheme задана функция, компонента для которой он задан не будет иметь соответсвующего дефолтного знаяения.

## Характеристики

По дефотлу прописана такая тема:
```js
{
    colorScheme, // задается в App
    defaultRadius: 'md', // дефолтный радиус компонент. Не все компоненты его используют.    
    globalStyles: () => ({ // настройки стилей
        body: {
            backgroundColor: backgroundColor, // автоматическое изменение цвета фона в зависимости от цветовой схемы. Задается в App.
        }
    }),
    ...mantineTheme // параметр из настроек проекта. Т.е. заданая тема будет заменять совпадающие дефолтные параметры.
}
```
Пример использования mantineTheme:
```html
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
<script>
    window.R = {
        states: {},
        env: {},
        params: {
            mantineTheme: {
                components: {
                    Image: {
                        defaultProps:{
                            radius:'md'
                        }
                    }
                }
            }
        },
        libs: {},
        utils: {},        
    }   
</script>
```
Пример использования mantineTheme c функцией:
```html
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1">
<script>
    window.R = {
        states: {},
        env: {},
        params: {
            mantineTheme: {
                components: {
                    Image: {
                        defaultProps: (theme) => ({
                            radius: theme.colorScheme === 'dark' ? 'xl' : 'md',
                        }) 
                    }
                }
            }
        },
        libs: {},
        utils: {},        
    }   
</script>
```
