# CSS

Все наши компоненты умеют принимать оба CSS параметра Noodl:
* CSS Style: просто прописываем стили в формате CSS.
* CSS Class: проставлять класс не нужно, т.к. в Mantine все компоненты имеют прописанные классы.

:::warning Не рекомендовано
Использовать CSS нужно только в случаях, когда нельзя задать стили средствами компоненты. Если нужно то лучше использовать Style, т.к. он локален.
:::

## Пример использования классов CSS Mantine

* Ищем Styles API для требуемого компонента, например для [Image](https://v6.mantine.dev/core/image/?t=styles-api)
* Вставляем Image.
* Добавляем компонент Noodl CSS Definition с таким содержанием:
```CSS
.mantine-Image-placeholder {
    background-color: brown;
}
```
* В результате, при отсутсвии картинки, Placeholder поменяет фон.
* Классы работают глобально. В этом примере, все компонентв Image поймают этот CSS.
