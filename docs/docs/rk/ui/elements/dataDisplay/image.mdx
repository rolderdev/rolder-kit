# Image

Отображает картинку.

Дефолтно занимает 100% ширины родительского контейнера, сохраняя пропорции. Пока не указана высота сохраняет пропорции.

Использует Placeholder, когда не указан источник или источник не возвращает картинку. Размер Placeholder равен заданым размерам картинки. Размер иконки Placeholder нужно подстраивать под размер картинки.

[Документация Mantine](https://v6.mantine.dev/core/image/)

## Конфигуратор

import Image from '../../../../../src/elements/dataDisplay/image'
