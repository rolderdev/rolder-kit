# fetch

Загружает данные одного класса.

## Параметры

* Db class: класс БД без версии.
* Filters: [фильтры Kuzzle](https://docs.kuzzle.io/core/2/api/koncorde-filters-syntax/clauses/).

```js
[{
    equals: {
        'content.name': "Петя"
    }
}]
```

* Sorts: сортировки.

```js
[
    {'_kuzzle_info.createdAt': 'desc'},
    {'content.name': 'asc'},
]
```

* Size: количество возвращаемых item.
